1° ação antes do calculo:

Antes de calcular o Delta, passamos por dois passos.
Em cada coeficiente, temos uma letra diferente, A = -2x, B = 2x², C = 1. Lembrando que só é possivel o B ser um coeficiente com X, caso não tenha, o coeficiente é considerado C.

2° ação antes do calculo:

O proximo passo também é feito em todos os calculos para encontrar o Delta. Usamos a base b²-4.a.c, com os coeficientes certos separados no passo 1, colocamos os respectivos nos locais certos de acordo com as letras. Após esses passos, podemos calcular o valor do Delta.
